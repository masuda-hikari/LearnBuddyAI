# LearnBuddyAI

**あなた専用のAI家庭教師 - いつでも、どこでも、あなたのペースで学習**

## 概要

LearnBuddyAIは、最先端のAI技術を活用したパーソナル学習プラットフォームです。従来の家庭教師の数分の1のコストで、24時間365日利用可能な個別指導を提供します。

## 特徴

- **パーソナライズ学習**: あなたの理解度に合わせて難易度を自動調整
- **いつでも質問**: 疑問が浮かんだらすぐにAIチューターに質問
- **進捗トラッキング**: 学習の進捗を可視化し、弱点を効率的に克服
- **スペースドリピティション**: 科学的に証明された間隔反復で記憶を定着

## はじめ方

1. アカウントを作成
2. 学習したい科目を選択
3. AIチューターとチャットを開始

```bash
# 開発環境セットアップ
cd backend
npm install
npm run dev
```
